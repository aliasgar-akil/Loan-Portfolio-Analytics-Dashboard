# Overview

The **Loan Portfolio Analytics Dashboard** is an interactive Power BI project developed to analyze the performance and risk of a consumer loan portfolio. It provides a comprehensive view of lending activities by combining portfolio-level KPIs, borrower demographics, loan characteristics, and credit risk metrics into a single reporting solution.

The dashboard enables users to monitor key lending indicators such as total loan amount, default rate, average credit score, and interest rates, while also exploring borrower segments based on income, age, education, credit score, and debt-to-income (DTI) ratio. Through Year-over-Year trend analysis, AI-powered visuals, and drill-through capabilities, users can identify risk patterns, evaluate portfolio performance over time, and investigate specific loan segments in greater detail.

The project demonstrates the complete Power BI development workflow, including data modeling, DAX calculations, time intelligence, interactive dashboard design, and business-focused reporting to transform raw loan data into actionable insights for decision-making.
